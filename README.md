🤖 JARVIS – Advanced Real-Time AI Personal Assistant (Python)

JARVIS is a powerful real-time AI personal assistant built in Python, capable of answering live queries, controlling Windows OS, executing keyboard commands, managing files, storing memory, and performing intelligent system automation — inspired by Iron Man’s JARVIS.

This project is designed for automation, speed, extensibility, and real-world desktop usage.

✅ Real-Time AI Interaction

Instant AI-powered responses

Text-based & optional voice commands

Async execution for low latency

Context-aware conversations

✅ Windows System Control

Open / close applications

Control volume & brightness

Access system utilities

✅ Keyboard & Mouse Automation

Execute keyboard shortcuts

Automated typing

Mouse movement & click automation

Task scripting support

✅ File & Folder Management

Open files instantly

Search directories

Quick file navigation
